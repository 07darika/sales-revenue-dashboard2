# sales-revenue-dashboard2
Professional Sales &amp; Revenue Analysis Dashboard using Power BI
Sales & Revenue Analysis Dashboard
Project Overview

This project is a professional Power BI Sales & Revenue Analysis Dashboard created using the Global Superstore Dataset.
The dashboard helps analyze business performance through interactive visualizations, KPI tracking, sales trends, product analysis, and customer insights.

Objectives
Analyze sales and revenue performance
Track important business KPIs
Identify top-performing products and categories
Understand regional sales distribution
Analyze customer segments and shipping modes
Generate business insights using interactive dashboards

DAX Measures Used
Total Sales
Total Sales = SUM(Superstore[Sales])
Total Profit
Total Profit = SUM(Superstore[Profit])
Total Orders
Total Orders = COUNT(Superstore[Order ID])
Profit Margin %
Profit Margin % =
DIVIDE([Total Profit],[Total Sales],0)*100
