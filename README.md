# Sales & Revenue Analysis Dashboard

## Project Overview
This project is a professional Power BI Sales & Revenue Analysis Dashboard created using the Global Superstore Dataset.

The dashboard helps analyze business performance through:
- Interactive visualizations
- KPI tracking
- Sales trend analysis
- Product analysis
- Customer insights

---

## Objectives
- Analyze sales and revenue performance
- Track important business KPIs
- Identify top-performing products and categories
- Understand regional sales distribution
- Analyze customer segments and shipping modes
- Generate business insights

---

## Tools & Technologies Used
- Power BI Desktop
- Microsoft Excel
- DAX (Data Analysis Expressions)

---

## Dataset Used
- Global Superstore Dataset

Dataset includes:
- Sales
- Profit
- Orders
- Product Categories
- Customer Segments
- Shipping Information
- Regional Data
